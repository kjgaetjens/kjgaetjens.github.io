Brief

Proposition 37 Map looks at broad voting patterns in California on Proposition 37 in 2012.

This map was created for a hypothetical scenario where I work in the field of electoral politics, and am specifically interested in ballot measures, where citizens vote directly to enact new laws. In this scenario, an agency partner in California has provided me with precinct level voting data – polygons matching the actual voting locations in the state, but this information is too noisy to notice true trends. To visualize broad voting patterns, I aggregated data to the County level and generated a map using the following steps.

The Counties layer was created using data provided by UC Davis Extension via the Fundamentals of GIS course. The PrecinctVotingDataSummarizedByCounty table was created by using ArcGIS's summarization tool on Precinct Voting Data. Total Votes and Yes Votes were summarized by the County field in the Precinct Voting Data source table. The resulting output, PrecinctVotingDataSummarizedByCounty, was joined to the Counties layer on Counties.FIPS_CODE and PreinctVotingDataSummarizedByCounty.COUNTY. This provided a more precise join than using a Spatial Join, as some precinct lines cross county lines.
